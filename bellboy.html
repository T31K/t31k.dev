Project Goal

As a flight crew, I have travelled to countless countries and myriad of hotels. Even so, sometimes I've encountered hotels which I've not travelled to. This app serves as a tool to flight crews/frequent flyers/leisure travellers alike to get an insider view of hotel rooms, their quirks and how to get the most out of them during your stay, long or short.

    • Project Achievements

  • Set up backend server with Express/Node to communicate from DB to the frontend
  • Incorporated MongoDB and Mongoose to CRUD data
  • Implemented user sign up and authentication using Passport.js
  • Designed interactive UI with EJS and styled with Bootstrap
  • Commited to Github for development, and Heroku for production
